# Realty Intel Pro V18 Model Card

## Intended Use
Buyer due diligence, listing claim verification, overpricing risk screening, and offer-preparation support.

## Not Intended Use
- Final appraisal
- Legal advice
- Investment guarantee
- Replacement for inspector, agent, lender, appraiser, attorney, or title company

## Model Stack
- Rule-based claim extraction
- Source verification framework
- Comparable-sales similarity scoring
- Hedonic adjustment scaffold
- Weighted factor scorecard
- Confidence and missing-source reporting
- Backtesting API contract

## Accuracy Requirements for Production
- MLS/RESO closed sale access
- Assessor data validation
- Permit and seller-disclosure ingestion
- Neighborhood-level walk-forward validation
- Error metrics by price band, property type, neighborhood, and sale month

## Key Metrics
- MdAPE / MAPE for valuation
- Interval coverage
- Claim verification precision/recall
- False-negative risk flags
- Citation coverage
- Freshness SLA
