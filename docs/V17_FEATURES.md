# V18 Feature Matrix

## Core User Flows
- Enter address or partial address
- Select dropdown suggestion
- Paste listing URL/source
- Paste full listing/ad/broker remarks
- Enter manual facts: price, beds, baths, sqft, year built, HOA
- Extract ad claims
- Run full V18 analysis
- Review fair-value range, overpricing verdict, claim conflicts, missing sources, hidden costs, investment model, factor scores, and generated report

## V18 Intelligence Layers
1. Input normalization
2. Address autocomplete fallback
3. Ad claim extraction
4. Claim verification
5. Property fact fusion
6. Synthetic comparable-sale engine
7. Hedonic adjustment scaffold
8. Confidence scoring
9. Uncertainty interval
10. Hidden ownership cost model
11. Rental/investment model
12. Natural hazard/risk scaffold
13. School/commute/zoning placeholders
14. Source audit trail
15. Missing critical source warnings
16. Backtesting dashboard contract
17. Buyer action plan and report generation

## Not Production Claims
This version is intentionally honest. It does not claim real appraisal accuracy until authoritative source connectors are activated.
